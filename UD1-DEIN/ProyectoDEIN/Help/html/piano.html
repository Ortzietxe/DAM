Crear Partituras

Para la creación de partituras basta con abrir la segunda pantalla desde el menú File> Nuevo. Esta segunda pantalla permite generar, imprimir y ejecutar una partitura.

Menu File

El menú File, consta de dos opciones que son las siguientes:


  • Nuevo
  • Ayuda

Nuevo(Alt-N): Muestra la ventana del editor de partituras.
Ayuda(Alt-A): Muestra la ayuda de la venta del simulador de piano.
